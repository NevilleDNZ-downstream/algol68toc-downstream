Name

tang - preprocess a Web 68 file

Summary

tang [-d] [-e] [-h] [-V] [-v] [-wdir] tang-file

Description

tang extracts the underlying Algol 68 code from a Web 68 file. The output is suitable for compilation.

The Web 68 file is arbitrarily broken into lines. Because tang regards a linefeed character as a space, this means that the meaning of the input is independent of its position. The meaning of text in a Web 68 file is controlled by Web 68 commands. See the Web 68 manual for further details.

tang takes the following options:

  • -d: switch on debugging. Output to tang.dbg
  • -e: output SALTU instead of SKIP
  • -h: output a concise usage message and exit
  • -V: do not output comments to the Algol 68 output
  • -v: output the version and exit
  • -wdir: add dir to the search list for Web 68 include files
